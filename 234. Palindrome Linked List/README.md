# 234. Palindrome Linked List

### Easy

---

Given the head of a singly linked list, return true if it is a palindrome.

## Example 1:

<pre>
Input: head = [1,2,2,1]
Output: true
</pre>

## Example 2:

<pre>
Input: head = [1,2]
Output: false
</pre>

## Constraints:

- The number of nodes in the list is in the range [1, 10^5].
- 0 <= Node.val <= 9
