# 231. Power of Two

### Easy

---

Given an integer n, return true if it is a power of two. Otherwise, return false.\
An integer n is a power of two, if there exists an integer x such that n == 2x.

## Example 1:

<pre>
Input: n = 1
Output: true
Explanation: 20 = 1
</pre>

## Example 2:

<pre>
Input: n = 16
Output: true
Explanation: 24 = 16
</pre>

## Example 3:

<pre>
Input: n = 3
Output: false
</pre>

## Constraints:

- -2^31 <= n <= 2^31 - 1
