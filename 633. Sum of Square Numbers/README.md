# 633. Sum of Square Numbers

### Medium

---

Given a non-negative integer c, decide whether there're two integers a and b such that a^2 + b^2 = c.

## Example 1:

<pre>
Input: c = 5
Output: true
Explanation: 1 * 1 + 2 * 2 = 5
</pre>

## Example 2:

<pre>
Input: c = 3
Output: false
</pre>

## Constraints:

- 0 <= c <= 2^31 - 1
